## Getting Started

Projeto criado para estudo de NEXTJS e Typescript

Para teste pode se utilizar o comando abaixo

```bash
npm run dev
# ou
yarn dev

E Abrir o endereço [http://localhost:3000](http://localhost:3000) no navegador para exibição do resultado.
```
